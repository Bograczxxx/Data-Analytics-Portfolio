# 01 — Analiza sprzedaży w Excel

## Opis projektu
Pierwsza analiza danych — nauka podstawowych narzędzi Excel/Google Sheets.

## Czego się nauczyłem
- Formuły: SUMA, ŚREDNIA, MAX, MIN, ILE.LICZB / SUM, AVERAGE, MAX, MIN, COUNT
- Logika warunkowa: JEŻELI / IF
- Wyszukiwanie danych: WYSZUKAJ.PIONOWO / VLOOKUP
- Tabela przestawna / Pivot Table
- Wykres przestawny / Pivot Chart

## Wnioski z analizy
- Laptop generuje największą sprzedaż (8 300)
- Styczeń był najlepszym miesiącem (5 000)
- Mysz ma najniższą sprzedaż (610)
